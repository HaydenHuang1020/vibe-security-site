## 责任与限制

### Vibe Security 的责任：

1. 在约定时间内完成检查并交付报告。
2. 按授权范围操作，不超出授权进行测试。
3. 对发现的漏洞和风险如实报告。
4. 修复套餐按确认的问题清单和工作量交付修复代码。

### Vibe Security **不**承担：

1. 不保证应用"绝对安全"——安全检查只能发现和修复已知风险，不能覆盖所有未知威胁。
2. 不对因第三方依赖、平台漏洞或客户自行修改代码导致的安全问题负责。
3. 不对检查完成后新引入的漏洞负责。
4. ¥399 体检套餐不包含代码修复。
5. 修复套餐中，新功能开发不属于漏洞修复范围。
6. 超出套餐约定范围的修复按单独报价执行。

### 客户的责任：

1. 确认自己有权授权对代码仓库和应用进行检查。
2. 提供准确的测试环境信息和凭证。
3. 检查完成后删除测试环境中的共享凭证。
4. 不要求 Vibe Security 对生产环境进行测试。
5. 未经书面授权不公开报告和漏洞细节。
